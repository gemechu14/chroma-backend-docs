Database Modules
================

CRUD modules provide tenant-scoped database operations used by API routes.

- ``src/crud/user.py`` — user lookup, authentication helpers, and staff persistence
- ``src/crud/tenant.py`` — tenant profile, locations, and subscription state
- ``src/crud/customer.py`` — customer records per tenant
- ``src/crud/product.py`` — global brands, lines, and products
- ``src/crud/tenant_product.py`` — tenant catalog configuration
- ``src/crud/inventory.py`` — stock levels and inventory transactions
- ``src/crud/formula.py`` — formula and visit persistence with stock deduction
- ``src/crud/subscription_plan.py`` — subscription plan catalog seeding and reads

ORM model definitions are documented under :doc:`Models and Services <models-services>`.
