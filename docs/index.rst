Dosecolor Backend Documentation
===============================

Welcome to the Dosecolor Backend API documentation.

Dosecolor is a FastAPI backend for multi-tenant salon operations, including authentication, inventory tracking in grams/mL, formula workflows, customer management, and subscription billing.
